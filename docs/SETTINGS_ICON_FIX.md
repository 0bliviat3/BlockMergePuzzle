# 🎨 설정 버튼 아이콘 수정 완료

## ✅ 문제 해결

### 문제:
```
❌ 이모지 (⚙️) 아이콘이 깨짐
❌ Unity에서 이모지 렌더링 불가
```

### 해결:
```
✅ "SET" 텍스트 사용 (명확하고 깨지지 않음)
✅ 간단한 톱니바퀴 그래픽 자동 생성
✅ 사용자 이미지 사용 가능 (선택 사항)
```

---

## 🎨 3가지 표시 방법

### 방법 1: "SET" 텍스트 (기본) ⭐ 추천

```
┌────────┐
│  SET   │  ← 간단하고 명확!
└────────┘

장점:
- 깨지지 않음
- 읽기 쉬움
- 언어 무관
```

**현재 기본 설정입니다!**

---

### 방법 2: 간단한 톱니바퀴 그래픽 (자동 생성)

```
┌────────┐
│   ⬤    │  ← 원형 + 톱니 4개
│  ╋╋╋   │
└────────┘

장점:
- 자동 생성
- 설정 의미 명확
- 이미지 필요 없음
```

**코드에 이미 포함되어 있습니다!**

---

### 방법 3: 커스텀 이미지 사용 (선택 사항)

```
┌────────┐
│  🖼️    │  ← 사용자 제공 이미지
└────────┘

장점:
- 원하는 디자인 가능
- 고품질 아이콘
```

---

## 🚀 사용 방법

### 기본 사용 (아무 설정 필요 없음):

```
1. SettingsManager 오브젝트 생성
2. Add Component → SettingsManager
3. Auto Create UI: ✓ 체크
4. Play

→ "SET" 텍스트 + 간단한 톱니바퀴 아이콘 자동 생성!
```

---

### 커스텀 이미지 사용하기 (선택 사항):

#### 1단계: 이미지 준비

```
- settings_icon.png (128x128 권장)
- Unity Project 창에 드래그
```

#### 2단계: Sprite 설정

```
이미지 선택
→ Inspector
→ Texture Type: Sprite (2D and UI)
→ Apply
```

#### 3단계: SettingsManager에 연결

```
SettingsManager 선택
→ Inspector
→ Use Text Button: ✗ 체크 해제
→ Settings Icon: settings_icon 드래그
```

#### 4단계: Play 테스트

```
→ 커스텀 아이콘 표시됨!
```

---

## 🎨 현재 버튼 디자인

### 자동 생성되는 디자인:

```
┌──────────┐
│          │
│   SET    │  ← 텍스트
│    ⚙     │  ← 간단한 톱니바퀴
│          │
└──────────┘

색상:
- 배경: 어두운 회색 (투명도 90%)
- 텍스트: 흰색
- 톱니바퀴: 흰색 + 회색
```

### 크기 & 위치:

```
크기: 100x100 픽셀
위치: 우상단 (20픽셀 여백)
```

   - 검색: "settings icon"
   - 무료 다운로드
   - PNG 형식 선택

2. **Font Awesome** (https://fontawesome.com)
   - 검색: "cog" 또는 "gear"
   - PNG 다운로드

3. **Material Design Icons** (https://materialdesignicons.com)
   - 검색: "settings"
   - SVG → PNG 변환

## 🔧 코드 커스터마이징

### 텍스트 변경:

```csharp
// SettingsManager.cs의 CreateSettingsButtonGraphic() 함수에서:

textComp.text = "SET";  // 원하는 텍스트로 변경
// 예: "⚙", "설정", "MENU" 등
```

### 톱니바퀴 색상 변경:

```csharp
// CreateSimpleGearIcon() 함수에서:

outerImage.color = new Color(1f, 1f, 1f, 0.3f);
// → Color.yellow, Color.cyan 등으로 변경
```

### 버튼 크기 변경:

```csharp
// CreateSettingsUI() 함수에서:

settingsButtonRect.sizeDelta = new Vector2(100, 100);
// → (120, 120) 또는 (80, 80)으로 변경
```

## 🆘 문제 해결

### 버튼이 안 보이면:

```
1. SettingsManager가 Hierarchy에 있는지 확인
2. Auto Create UI가 체크되어 있는지 확인
3. Play 모드 재시작
```

### 아이콘이 이상하게 보이면:

```
1. Use Text Button: ✓ 체크 (텍스트만 사용)
2. 또는 커스텀 이미지 사용
```

### 텍스트가 잘리면:

```
버튼 크기 증가:
settingsButtonRect.sizeDelta = new Vector2(120, 120);
```
